#### Main goals

The 3 main goals in this course are:

*   Learning how to use a library
*   Making a datavisualisation from external data
*   Learning d3

    ([**class 2**][c2])

    ([**class 2**][c2] and [**class 4**][c4])
8.  <a name="subgoal-8"></a> Load external data
    ([**class 3**][c3] and [**class 4**][c4])

    ([**class 6**][c6])
