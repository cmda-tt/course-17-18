Bar chart

Dit document gaat over het uitleggen van mijn bar chart.

Background

Het gaat om de goodness van de kopjes koffie die ik afgelopen 2 weken heb gedronken. Elk kopje koffie heb ik een cijfer gegevens en het aantal per cijfer weergegeven.

For more information: (https://d3js.org/)

Data

Dit zijn de 2 type data die ik heb toegepast:

  • Scale - Cijfer dat ik heb geven om het kopje te beoordelen van 1 tot 10 op de y-as
  • Amount - Aantal kopjes dat bij het cijfer hoort op de x-as

Features

  • D3-DSV
  • D3-format
  • D3-scale
  • D3-array
  • D3-axis
